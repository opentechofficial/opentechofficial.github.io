Neo OS

Hello, this page is to maintain a documentation on the ongoing project of creating Neo OS, a new redesigned, refined OS for today's world.

Main Ideas

the following are Ideas we set forth for this OS:

  • Database Filesystem
  • Tag Based Storage
  • HTML/CSS based Apps
  • Redesigned Desktop Environment
  • Message Bus Daemon
  • OS-only file reading
  • App-Locks
  • VM code run
  • P2P App-Store
  • Dat- Compatible Browser
  • Universally Adaptable OS
